# 🚀 快速开始指南

## 预览新版本

### 方式一：本地预览（推荐）

1. **启动本地服务器**
   ```bash
   cd /home/engine/project
   python3 -m http.server 8080
   ```

2. **在浏览器中打开**
   ```
   http://localhost:8080
   ```

### 方式二：直接打开文件

```bash
# 在浏览器中直接打开 index.html
open index.html  # macOS
start index.html # Windows
xdg-open index.html # Linux
```

> ⚠️ 注意：直接打开文件可能导致某些功能受限（如CORS限制）

## 💡 使用建议

```bash

# 或使用 Node.js
npx http-server -p 8080

# 或使用 PHP

```

### 浏览器推荐
- Chrome 90+（最佳体验）
- Firefox 88+
- Safari 14+
- Edge 90+

- 桌面：1920x1080
- 平板：768x1024
- 手机：375x667
